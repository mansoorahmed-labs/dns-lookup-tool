# 🌐 DNS Lookup Tool

A Python-based DNS Lookup Tool that retrieves DNS records to support network troubleshooting, infrastructure enumeration, and cybersecurity reconnaissance.

---

## 🚀 Features

- ✅ Resolve IPv4 Address
- ✅ Retrieve A Records
- ✅ Retrieve AAAA Records
- ✅ Retrieve MX Records
- ✅ Retrieve NS Records
- ✅ Retrieve CNAME Records
- ✅ Error Handling
- ✅ DNS Enumeration
- ✅ Supports Network Reconnaissance

---

## 🛠️ Technologies Used

- Python 3
- Socket Library
- dnspython

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/mansoorahmed-labs/dns-lookup-tool.git
```

Navigate to the project:

```bash
cd dns-lookup-tool
```

Install the required dependency:

```bash
pip install dnspython
```

---

## ▶️ Usage

```bash
python3 dns_lookup_tool.py
```
